Worksheet 6

String formating

三种格式会输出相同的结果

f-string 格式是print(f"something")，引号内可以用{val}来输出某个值（变量等）

单引号（''）和双引号（""）可以随意互相替换，但要注意字符串内的标点，避免字符串“断掉”，如果句内包括了引号，有三种解决方案，一是利用另一种引号作为限定符，二是在句中引号前使用\会得到正常的引号输出，三是使用三重引号

三重引号提供了多行输出的功能，区别于普通引号换行会出现编译错误，三重引号中的换行会在输出中显示出来，并得到正常输出。（单双引号也可以随意替换，任意组合）

在f-string中，使用{float:n.mf}（其中n和m为指定常量）右端在第n个字符位置对齐（如果数字长度小于这个数字的话，数字长度大的会一直往右把数输出完，或者说会在数字左边加上max{0,n-len(val:.fm)}个空格）同时保留m位小数，如果在n前面加上0，则用0来取代前面所提的空格。

同样的，对于字符串，我们可以用s来代替f，也就是{string:n.ms}，区别是字符串默认左端对齐，而浮点数默认右端对齐，同时字符串没有把空格改成0的操作（如果要用某字符替代空格，可以在n前加上字符+对齐符号，类似:a<n，此时空格可被替代，浮点数也可以用同样的方式输出（替位符不一定要是数字/字母））

对于s，目前测试下来是可省略的，而f若省略功能则变成保留有效数字，若将点号后面省略则是单独的格式修改（不会改变初始值），冒号后整体被称作format specifier，点号后的数字叫做Precision，最后的字母叫做format code

在n前输入<或^或>可以分别对输出进行居左，居中，居右对齐的设置

:g表示general format，会根据数字自动调整数据类型，例如2/2:g会输出1，:d输出为普通十进制整型，:c会输出字符，可以输出一个整数在Unicode中对应的符号。

在前面提到的n和m处如果要使用变量，需要在表达式外额外再添加一层大括号。

我自己的总结大概是对于{something:a<n.ms}其中最后一个字母设置format code，主要用来辨识数据类型（例如c,d,f,g,s），.m针对该数据类型进行截取（f：保留小数点后位数，s：保留字符串字符数，省略则根据原始数据类型，但对于float是保留有效数字），n代表对齐格式的右端位置，<,^,>三种对齐格式，a位置则是空格替换（在float类型下可以直接带0，其他情况一定要在没有进行precision，同时有对齐格式的基础上添加，但是可以替换任意字符（至少包括字母数字和$））

.getpixel(address)返回图片某个像素的值，address是(column, row)，坐标从(0, 0)开始

对于返回值，根据图片colour-mode返回不同

  • For a black and white image (mode "1"), it will be 0 for black, or 1 for white.
  • For a greyscale image (mode "L"), it will be a number between 0 and 255 (inclusive): 0 for black, 255 for white.
  • For an RGB image (mode "RGB"), it will be a 3-tuple of values. This tuple will contain three values, all between 0 and 255 (inclusive). The first value will be the amount of red in the pixel, the second the amount of green, and then third the amount of blue. In each channel, 0 represents absence of that colour and 255 represents the full amount of that colour.

Data science

involve 4 steps：

  1. data wrangling: cleaning up the raw data to remove anomalies, integrate data from multiple sources into a single format, and generally make it easier to use; this is often one of the most time-consuming and tedious parts of data science, but is an essential first step, as "garbage in, garbage out"
  2. data visualisation: generating visual representations of the data in different ways, both as a pre-step to "get to know" the data (based on different breakdowns of the data) and as a post-step to better understand what has been gleaned about the data
  3. data analysis: statistical analysis of the data to "model" some aspect of it, in terms of unearthing patterns in the data, or predicting things about new events/behaviours.
  4. empirical evaluation: measurement of the "correctness" of a given data analysis model to the data, in terms of how well it is expected to predict future events/behaviours, or how well it "fits" the data.

Exception Handling

就是防止exceptional case导致崩溃

  1. 直接用if-statement，缺点是太general

  2. 在function内在出状况的情况下使用raise Exception(<error statement>)创建一个exception object（实际上也并不是非要自己创建，碰到问题程序自己就会报错）

    然后在调用的时候使用try except，会把error statement赋给as后定义的变量

  3. 可以用多个except来处理不同的error，此时顺序也会有影响

  4. 还可以添加else语句，如果不报错就会接着执行

  5. 添加finally语句，会在退出try语句后触发，即使是用了break，return也会触发

  6. assert(condition)当条件为True，会有AssertionError

exit()不再进行后续操作

对于报错，如果程序没有后续处理，程序就会自己中断并报错

S –> NP VP VP –> IV | TV PN NP –> N | D N IV –> runs | sits PN–> john | mary N –> squirrel | cat | mouse | dog | tree TV –> chases | catches | tells | sees | eats D–> the | a

This grammar states that a sentence (S) is composed of a noun phrase (NP) followed by a verb phase (VP). VP may be either an intransitive verb (IV) or a tranvisitve verb (TV) followed by a proper noun (PN). NP is either a noun (N) or a determiner followed by a noun (D N). For each class (so called "non-terminal") such as PN,N, TV, IV, D we provide a list of words (terminals) writter in lowercase (cat, tells, etc.). Using this grammar, we can generate sentences such as the mouse eats john or the tree runs as well as check whether a sentence is acceptable given the grammar (i.e. can be produced using the grammar rules). For instace, a sentence a cat sees mary is valid since is can be produced/parsed using the following grammar rules [[ a.D cat.N].NP [sees.TV mary.PN].VP].S (i.e. we are able to reach S) whereas a sentence a cat sits mary is not (there is no rule that has IV PN in the righthand side).
